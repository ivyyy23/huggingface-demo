# Hugging Face Demo
A small Python demo using Hugging Face Transformers pipelines to summarize text and predict sentiment.

## Features
- Summarizes long text using summarization pipeline.
- Analyzes sentiment (positive/negative) of the summary using sentiment-analysis pipeline.
- Demonstrates chaining of pipelines.

## How to Run
1. Install dependencies:
```bash
pip install transformers torch
```

2. Run the script:
```bash
python test_pipelines.py
```
